
## 🐍 Python do Zero: 10 Lições Práticas
Bem-vindo ao curso Python do Zero ao Primeiro Script! Este repositório contém 10 pequenos programas projetados para ensinar os fundamentos da programação de forma progressiva e prática.

## 🚀 Objetivo
Aprender a sintaxe básica do Python através de desafios reais, indo desde o clássico "Olá Mundo" até a criação de funções personalizadas.

## 📚 Conteúdo das Lições
Lição	Arquivo	O que você vai aprender?
01	desafio_01.py	Saída de dados com print().
02	desafio_02.py	Variáveis e tipos de dados (str, int, float).
03	desafio_03.py	Entrada de dados com input() e conversão de tipos.
04	desafio_04.py	Operadores aritméticos e cálculos simples.
05	desafio_05.py	Controle de fluxo com if/else.
06	desafio_06.py	Armazenamento de coleções em Listas.
07	desafio_07.py	Repetições automáticas com o laço for.
08	desafio_08.py	Loops baseados em condições com while.
09	desafio_09.py	Organização de dados com Dicionários (Chave:Valor).
10	desafio_10.py	Modularização e reutilização de código com Funções.

## 🛠️ Como usar este repositório
Pré-requisitos: Ter o Python instalado em sua máquina.
Clonar o projeto:
bash
git clone https://github.com
Use o código com cuidado.

Executar uma lição:
Navegue até a pasta do projeto e digite:
bash
python desafio_01.py
Use o código com cuidado.

## 📝 Desafio Final
Após completar as 10 lições, tente criar um script que misture Listas, Loops e Funções (ex: um sistema de cadastro de produtos ou uma lista de tarefas).
